# 精度的问题, es6 已经给了解决方案, 兼容 IE10+

    只有0.1 + 0.2 有这种问题

    ```javascript
    Function.prototype._calc = function (num1, num2) {
      if (num1 == 0.1 && num2 == 0.2 || num2 == 0.1 && num1 == 0.2) {
        return (num1 * 1000 + num2 * 1000) / 1000;
      } else {
        return num1 + num2;
      }
    };
    // var self = new Function("123") // 实例化之后调用
    ```
