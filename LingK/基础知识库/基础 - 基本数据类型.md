# 常用的Api
## Number
### 取整
```shell
	Math有三种取整的方式:
	1. ceil: ceiling天花板, 向上取整, 并非绝对值, 从数值上向上取整
	2. floor: floor地板, 向下取整, 并非绝对值, 从数值上向下取整
	3. round: 四舍五入, 一位一位的进行取整, 最后一位round → 倒数第二位round
```
